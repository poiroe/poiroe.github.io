换域名啦，github提供的免费me域名。

教程
  • github
  • 域名

github提供的学生开发包，内含me域名，DigitalOcean优惠券SSL证书等。

申请方法

一、edu域名申请

freedns官网：

注册:http://freedns.afraid.org/signup/


填写帐号信息，验证码后发送激活邮件。



登录后点击左边的Registry然后搜索.edu域名



填写第三方dns解析

域名申请完毕，接下来教大家搭建域名邮箱，这里选择的是google，网易和阿里云的都要验证一级域名权限。

二、域名邮箱搭建

google官网：

google域名邮箱注册:https://www.google.com/intx/zh-CN/work/apps/business/

啊！懒癌发作了,不想重新注册演示了，大家进入页面后点击免费使用填写帐号信息注册，然后根据提示在dns托管商那添加CNAME、MX等记录。

三、Student Developer Pack申请

github官网:

申请:https://education.github.com

登录github帐号，没有的先注册然后在个人设置里绑定前面申请的edu邮箱。

登录后点击右上角Request a discount填写申请信息。

提交完毕后耐心等待审核期间不杀生，不淫欲，不妄语，不饮酒多做好事攒RP


免费资源不可多得，github初衷是让学生能够免费使用最好的开发工具，让他们可以通过动手来学习，so 各位切勿滥用。
